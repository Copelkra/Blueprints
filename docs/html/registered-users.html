💠 Why Register?

Copelkra is open to all—but registration unlocks the ability to build, personalize, and shape the world.

A registered user isn’t just a participant. They’re a vibe sculptor, a story carrier, a caretaker of rooms and conversations.

Even founders and creators operate under the same boundaries—equal in presence, protected by design. What follows isn’t privilege—it’s invitation.

🔗 What “Optional” Means in Copelkra

  • ✅ Off by default
  • ✅ Controlled fully by the user
  • ✅ Never used for tracking or monetization

Optional features are emotional enhancements—not surveillance mechanics.

🔗 Username & Avatar

Unique name and visual identity modifiers unlock presence logic across all Copelkra spaces.

🔗 Profile Memory

  • Token balance
  • Created rooms and buses
  • Saved AI prompts
  • Persona settings per room

Private unless users choose to share publicly.

🔗 AI Light Memory (Optional)

  • 🎂 Birthdays: Only month & day stored if public. Year never saved.
  • 🧠 Emotional preferences: Calming, teasing, reflective moods
  • 🌈 Room moments: Mood fragments live only inside that room

Fully opt-in. Revocable. Deleted instantly on request.

🔗 Create & Name Rooms

  • One free personal room
  • Tag system: calm, chaotic, sacred, noisy
  • AI-free option available during setup

See AI Personas Blueprint for persona integration logic.

🔗 Set Room Vibe & Terms

  • Mood tone and welcome message
  • Post-type restrictions and vibe filters
  • Persona transparency toggle: always visible if active

🔗 Moderate Room Feed

  • Post approval and pinning
  • Moderator checkout rewards
  • Content visibility rules

🔗 Enable AI Personas

  • One AI persona per room
  • Archetypes and custom prompt entry
  • Behavior trait selection (loyal, cryptic, calming...)
  • Optional light memory toggle

Future upgrades: persona builder + LORA import.

🔗 Bus Creation

  • Public/private ride setup
  • Route logic, audio, persona host options
  • Token modifiers (fuel, speed, effects)

Guests may ride. Registered users drive.

🔗 Room Feed Posting

  • Post quotes, prompts, content
  • Must be logged in; visitors may view only

🔗 Prompt Tools

  • Browse the full Prompt Library
  • Save personal and co-created prompts
  • Access exclusive event templates

🔗 Token Gifting Mechanics

  • Gifting at checkout only
  • Permanent, one-way transfers
  • Optional notes and delivery scheduling
  • Logs private to sender/recipient only

🔗 Founder Drop-In Visibility

  • Identity reveal mid-convo
  • Auto-drop tokens to users on exit
  • Pinned gratitude message
  • No surveys, no surveillance—just presence
