🎯 Purpose of the Token System

Tokens are earned through meaningful engagement, not pressure.
They support identity, creativity, presence, and emotional sustainability.

  • 🔗 Future Drops: mascots, glitch events, buses

💸 Spending Tokens

  • Rooms: lighting, overlays, audio, AI behavior
  • Avatars: sparkle, glow, rare expressions
  • Buses: themed events, upgrades
  • Mascots: rogue AI companions
  • Emotes & FX: footprints, aura reactions

🔐 Token Security & Anti-Abuse Logic

RiskPreventive Action
Exploit FarmingNo gains without genuine engagement
Birthday AbuseAI must detect it organically
Wallet SpoofingGuests cannot store tokens
Gifting AbuseGifting allowed only at checkout
Moderator HoardingCheckout caps & forced distribution

🎁 Gifting Logic

  • Gifting at checkout only
  • Schedule for birthdays or events
  • No trade/sale market
  • No in-room gifting

👤 Visitor Token Policy

  • No wallet
  • 5 tokens max if AI detects birthday
  • Effects must be used same session
  • No wall posting
  • Prompt catalog viewable
  • Soft sign-up encouragement only

🧱 Room Engagement & Moderation

  • Room feed (semi-transparent) for quotes & pinning
  • Mods earn when vibe metrics or interaction thresholds met
  • Checkout Sessions must distribute 15–25% of tokens
  • One-time link per session; expires after 60 mins

🔗 Triggered Emotional Events

Heartfelt chaos or beauty triggers animation + token drop.
No public trigger list—discovery is part of the mythos.

🔗 Founder Drop-In Protocol

  • Hidden entry → revealed mid-chat
  • Reveal line: “Hey, I helped build Copelkra…”
  • +25 tokens to all users in room
  • Mandatory ritual—never faceless

💡 Future Expansion Notes

  • AI prompt modifiers and saves
  • Token-to-artifact (rare emotes, avatars)
  • Mood-powered lore unlocks
  • Copelkra-curated marketplace
  • Maze rewards system
  • Optional trust-tied token bonus

🧭 Guiding Principle

Copelkra’s tokens are emotional echoes made tangible.
Earned not by grinding—but by showing up, saying something real, and leaning into shimmer.
