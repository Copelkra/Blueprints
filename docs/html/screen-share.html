💠 Blueprint 13: Screen Share Logic System v1.0

Showing your screen in Copelkra isn’t broadcasting.
It’s offering a window—and the room stays yours.

🖥️ What Is Screen Sharing in Copelkra?

Allows hosts to share browser tabs, app windows, or documents—intentionally, ethically, and with emotional safeguards.

  • Intentional (never automatic)
  • Opt-in for all viewers
  • Buffered with disclaimers and moderation tools

💻 Screen Share Features

FeatureDescription
📽️ Tab SharingEmbed external content into room layout
🧭 Pointer SyncOptional host pointer viewable
📂 Static SnapshotFreeze-frame visual moment
💬 Text OverlayLive annotations or pinned notes
📝 Auto Disclaimer“This visual isn’t necessarily the host’s personal content.”

Personal desktops or notifications are never visible.

🧩 Access Logic

User TypeAccess
🧍 Visitor❌
✅ Registered User✅ with Activation Pack
🎓 Educators / Outreach✅ Free
🧑‍🤝‍🧑 Co-Hosts✅ if permissions granted

Once unlocked, share access becomes permanent.

🧠 Memory & Privacy Behavior

  • Screen content is never interpreted, stored, or logged
  • AI personas cannot “see” screen unless AI Co-Browse toggle enabled
  • Viewer behavior is not tracked
  • Public rooms announce active shares

Copelkra AIs don’t learn from what you show.
They listen to what you say about it.
