🧍 What Is an Avatar in Copelkra?

Avatars are not tied to data, gender, or identity. They are emotional vessels.
Some sparkle. Some sit quietly. Some are socks.

They exist only where they’re needed, never getting in your way unless you want them to.

  • 🎧 Playback when user clicks avatar (optional toggle)
  • ☁️ Quip types: pre-set sounds, user recordings, uploaded stingers
  • 🔁 Reusability: saved in profile library
  • 👁️ Reaction emotes: floating emoji on trigger
  • 🔏 TOS enforcement: violation quips auto-muted & moderated
  • ⚙️ Trigger limits: once per minute per viewer
  • 🎭 Persona masking: cloaked avatars still play quip on direct tap
